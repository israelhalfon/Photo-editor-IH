# PhotoEditor Pro 📸

A modern, web-based photo editing application that allows you to upload and edit photos with various filters and adjustments.

## Features

### 🎨 Photo Editing Options
- **Black & White**: Convert photos to elegant grayscale
- **Chromatic**: Enhance colors with vibrant saturation
- **Paint**: Apply artistic paint-like effects
- **Enhanced**: Boost contrast and vibrancy for professional look

### ⚡ Adjustment Controls
- **Brightness**: Adjust image brightness (-100 to +100)
- **Contrast**: Control image contrast (-100 to +100)
- **Saturation**: Modify color saturation (-100 to +100)

### 🚀 User Experience
- **Drag & Drop Upload**: Simply drag photos into the upload area
- **Click to Browse**: Traditional file selection method
- **Real-time Preview**: See changes instantly as you edit
- **Download Edited Photos**: Save your creations as PNG files
- **Reset Function**: Quickly revert to original settings
- **Responsive Design**: Works perfectly on desktop and mobile devices

## How to Use

1. **Upload a Photo**:
   - Drag and drop an image file into the upload area, or
   - Click the upload area to browse and select a file

2. **Choose a Filter**:
   - Click on any of the filter buttons (Original, Black & White, Chromatic, Paint, Enhanced)

3. **Fine-tune with Adjustments**:
   - Use the sliders to adjust brightness, contrast, and saturation
   - Changes apply in real-time

4. **Save Your Work**:
   - Click the "Download" button to save your edited photo
   - Files are saved as PNG format with timestamp

5. **Start Over**:
   - Click "New Photo" to upload a different image
   - Click "Reset" to clear all adjustments

## Features in Detail

### Filters
- **Black & White**: Uses luminance formula for accurate grayscale conversion
- **Chromatic**: Enhances RGB channels for vivid colors
- **Paint**: Applies edge detection and color blending for artistic effect
- **Enhanced**: Increases contrast and vibrancy for professional results

### File Handling
- Supports all common image formats (JPEG, PNG, GIF, WebP)
- Automatic image resizing for optimal performance
- Maintains aspect ratio during resizing

### UI/UX
- Clean, modern interface with gradient backgrounds
- Smooth animations and transitions
- Intuitive drag-and-drop functionality
- Responsive grid layout for mobile compatibility
- Toast notifications for user feedback
